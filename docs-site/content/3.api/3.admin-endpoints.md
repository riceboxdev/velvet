# Admin Endpoints

These endpoints require an API Key for authentication.

**Header**: `X-API-Key: wl_xxxxx`

## Get Waitlist

**GET** `/api/admin/waitlist`

Get waitlist details with statistics.

---

## List Signups

**GET** `/api/admin/signups`

List all signups for your waitlist.

### Query Parameters

| Parameter | Description |
|-----------|-------------|
| `limit` | Max results (default: 50, max: 500) |
| `offset` | Pagination offset |
| `status` | Filter: `waiting`, `verified`, `admitted` |
| `sortBy` | Sort field: `position`, `created_at`, `referral_count`, `priority` |
| `order` | `ASC` or `DESC` |

---

## Admit User

**PATCH** `/api/admin/signups/:signupId/offboard`

Accept a user off the waitlist.

---

## Remove Signup

**DELETE** `/api/admin/signups/:signupId`

Remove a signup from the waitlist.

---

## Webhooks

### List Webhooks

**GET** `/api/admin/webhooks`

### Create Webhook

**POST** `/api/admin/webhooks`

```json
{
  "url": "https://your-server.com/webhook",
  "events": ["new_signup", "offboarded"]
}
```

### Webhook Events

| Event | Description |
|-------|-------------|
| `new_signup` | New user joins the waitlist |
| `offboarded` | User was admitted |
