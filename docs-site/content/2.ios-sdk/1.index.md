## Installation

```
https://github.com/riceboxdev/Velvet
```

Then import in your Swift files:

```swift
import Waitlist
```

## Configuration

Create a `WaitlistConfig` with your credentials:

```swift
let config = WaitlistConfig(
    apiKey: "your-api-key-here",
    waitlistId: "your-waitlist-id-here"
)
```

::callout{icon="i-lucide-info" color="blue"}
Get your API key and waitlist ID from the [Velvet dashboard](https://velvet.app).
::
