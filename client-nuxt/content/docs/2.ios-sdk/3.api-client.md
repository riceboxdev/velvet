# API Client

For complete control over the UI, use `WaitlistClient` directly.

## Basic Usage

```swift
let client = WaitlistClient(config: config)

Task {
    do {
        let response = try await client.signup(email: "user@example.com")
        print("Position: \(response.position)")
    } catch {
        print("Error: \(error)")
    }
}
```

## Methods

### signup

Add a user to the waitlist:

```swift
let response = try await client.signup(
    email: "user@example.com",
    metadata: ["source": "ios_app", "version": "1.0.0"]
)
```

Returns: `SignupResponse` with `position`, `message`, `referralCode`

### hasJoined

Check if this device has already joined:

```swift
if client.hasJoined() {
    // Show confirmation instead of form
}
```

### checkStoredStatus

Check the status of a previously signed-up user:

```swift
if let status = try await client.checkStoredStatus() {
    if status.accepted {
        navigateToMainApp()
    } else {
        print("Position: \(status.position)")
    }
}
```

### getStats

Get waitlist statistics:

```swift
let stats = try await client.getStats()
print("Total signups: \(stats.totalSignups)")
print("Waitlist name: \(stats.waitlistName)")
```

### getPosition

Check a specific user's position:

```swift
let position = try await client.getPosition(email: "user@example.com")
print("Position: \(position.position)")
print("Accepted: \(position.accepted)")
print("Referral count: \(position.referralCount ?? 0)")
```
