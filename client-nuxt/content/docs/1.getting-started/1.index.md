# Introduction

Velvet is a hosted SaaS platform that makes it easy to add waitlist functionality to your iOS app. With just a few lines of code, you can integrate a complete waitlist signup flow.

## What is Velvet?

Velvet provides:

- **iOS SDK** - Swift Package with pre-built SwiftUI components
- **REST API** - Full-featured API for custom integrations
- **Web Dashboard** - Admin interface to manage waitlists and signups

## How It Works

1. **Sign up** at [velvet.app](https://velvet.app) and create a waitlist
2. **Get credentials** - your API key and waitlist ID
3. **Integrate** - add the iOS SDK or use the REST API
4. **Manage** - view signups and accept users from the dashboard
