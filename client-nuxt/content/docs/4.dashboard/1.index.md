# Dashboard

The Velvet dashboard at [velvet.app](https://velvet.app) provides a web interface for managing your waitlists.

## Features

### Waitlist Management

- Create and configure multiple waitlists
- Set waitlist name, description, and status
- Regenerate API keys

### Signup Management

- View all signups in a table
- Sort by position, date, or referral count
- Search and filter signups
- Accept users off the waitlist
- Remove signups

### Analytics

- Total signup count
- Signup trends over time
- Referral statistics

### Settings

- Customize branding colors
- Configure webhook notifications
- Manage account settings

## Getting Your Credentials

1. Log in to the dashboard
2. Select your waitlist (or create a new one)
3. Go to **Settings**
4. Copy your **API Key** and **Waitlist ID**

Use these credentials to configure the iOS SDK or make API requests.
